The Challenges

Day 01 Challenge

  • 1) Limit the total width of the .intro-content to about half of it's parent
  • 2) Stop the text from overflowing out the bottom at small screen widths
  • You may modify the HTML if needed

Day 01 Starter Code

          
          .container {
            background: #23424a;
            color: white;
          
            width: 80%;
            margin: 0 auto;
          
            padding: 2em;
            height: 300px;
          }

My Solution

          
          .container {
            background: #23424a;
            color: white;
          
            width: 80%;
            margin: 0 auto;
          
            padding: 2em;
            /* deleting height: 300px 
            stops text from overflowing on the bottom 
            of the containter on smaller screens */
          }
          
          /* limit content to 50% of the container */
          .intro-content {
            max-width: 50%;
          }
