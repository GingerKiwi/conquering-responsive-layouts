About This Site

This site is my work from Kevin Powell's Conquering Responsive Layouts 21 Day Challenge.

Now in May 2023, I'm starting it again from the beginning. It's a nice way to keep up my css and UI design skills as I work through the more advanced javascript and react modules in the Scrimba Frontend Developer Career Path. This time I'm also creating this site and doing daily git dev branches then merging them with main.
