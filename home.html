About This Site

This site is my work from Kevin Powell's Conquering Responsive Layouts 21 Day Challenge.

I started this course in December, 2021 then got covid for the second time. It takes me 6 to 8 months to recover from it. So I didn't finish the course. Now in May 2023, I'm starting it again from the beginning. It's a nice way to keep up my css and UI design skills as I work through the more advanced javascript and react modules in the Scrimba Frontend Developer Career Path. The Challenges

Day 01 Challenge

  • 1) Limit the total width of the .intro-content to about half of it's parent
  • 2) Stop the text from overflowing out the bottom at small screen widths
  • You may modify the HTML if needed

Day 01 Starter Code

          
          .container {
            background: #23424a;
            color: white;
          
            width: 80%;
            margin: 0 auto;
          
            padding: 2em;
            height: 300px;
          }

My Solution

          
          .container {
            background: #23424a;
            color: white;
          
            width: 80%;
            margin: 0 auto;
          
            padding: 2em;
            /* deleting height: 300px 
            stops text from overflowing on the bottom 
            of the containter on smaller screens */
          }

          /* limit content to 50% of the container */
          .intro-content {
            max-width: 50%;
          }

Day 03: Challenge 02

You can view the starting layout here. The goal is to limit the width of all the text, especially on larger screen sizes. Both the text in the dark background and light background must line up on the left. The dark background must stretch across 100% of the screen.

Kevin's Notes: You are allowed to modify anything I've provided, including the HTML. Also, I provided a little more than intended in the CSS, but in the next challenge I'm providing you with nothing at all as a starting point, so a little extra help in this one won't hurt ðŸ˜….

Starting Code

          
            
* {
  box-sizing: border-box;
}

body {
  margin: 0;
  font-family: sans-serif;
}

.container {
  width: 80%;
  max-width: 750px;
  margin: 0 auto;
  background: #23424a;
  color: white;
  padding: 50px 0;
}

.intro-content {
  width: 50%;
}

My Solution

  • Put the bottom section "more content" inside a section tag then styled the section
  • Set a max-width on the section tag to 750px
  • Set padding for both section and intro-content to "padding: 3rem 4rem; "
  • Changed the intro-content class to stretch 50% with a max-width of 375px. This is a max-width of 50% of 750px set for the text in the section tag
          
            
* {
  box-sizing: border-box;
}

body {
  margin: 0;
  font-family: sans-serif;
}

section {
  padding: 3rem 4rem;
  max-width: 750px;
}

.container {
  width: 100%;
  margin: 0 auto;
  background: #23424a;
  color: white;
  padding: 3rem 4rem;
}

.intro-content {
  width: 50%;
  max-width: 375px;
}
