Day 04: Viewport Units - vh, vw, vmin, vmax

There's no challenge for day 04. The task is to become familiar with viewport units. Here's some takeways from the video:

  • The difference between vw (or vh) and percentages is that vw and vh are relative to the viewport whereas percentages are relative to the parent element.
  • There are accessibilty issues when using viewport units for font size.
  • 100vh is useful for hero sections. It will make them cover the entire viewport from top to bottom. However, there can be overflow issues on mobile, so it's best to use media queries and keep 100vh for larger screensizes.
  • 10vh for padding will adjust the padding based on the viewport height. This is useful for padding for elements like hero sections
  • vw can be useful for font size for headings. Doing so can make headings very responsive with little to no need for media queries.
  • Using vw for paragraph text runs into readability issues on smaller screens, and layout issues on larger screens.
